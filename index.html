ABOUT

Welcome to my website. My name is Adam Zaidi, and I love to program. This website is a portfolio
of different projects I have and are currently working on, from scroller games to fractals. All of
these projects are made with the coding languages p5js, processing, and javascript. The first two
are variations of javascript. I also code in html and css, and this website is completely handmade.

INFO
  • You can't play the games on a phone or tablet. If you would like to try out the games,
    please use a laptop or desktop computer.
  • To play many of these games, you need to click on the game so that you see a thin
    outline around the square. If you don't, you won't be able to play the game.
  • Many of these project are works in progress. I continue to update them and work out bugs.
