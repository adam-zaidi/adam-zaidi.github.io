FOURIER AND SINE WAVES

Explanation

This simulates a Fourier Series. As you slide the slider further, it adds
more circles and starts to better approximate a square wave. If you slide
the first slider all the way back, it will be a basic sine wave with only
the one main circle. The second slider changes a different variable, so
move that one to change the shape of the wave. If you slide it all the
way back, you will see a sawtooth wave.
